image
(image {:keys [width height src x y sx sy], :as props} & childs)
Image primitive, can be nested.

:param width: Text holder width - number.
:param height: Text holder height - number.
:param x: Text holder upper left corner x coordinate on parent.
:param y: Text holder upper left corner y coordinate on parent.
:param sx: Source image x, useful for cutting sprites.
:param sy: Source image y, useful for cutting sprites.
:param *childs: Primitives rendered inside the rectangle.

Usage:

```
(image {:width 100
        :height 200
        :src "/bird.png" ; absolute path to image
        :sx 20 ; x on source image, usable for cutting sprites
        :sy 30 ; y on source image, usable for cutting sprites
        :x 10
        :y 20}
  (rectangle {:color "#00ff00"
              :x 50
              :y 50
              :x 10
              :y 10}))
```
rectangle
(rectangle {:keys [width height color x y], :as props} & childs)
Rectangle primitive, can be nested.

:param width: Rectangle width - number.
:param height: Rectange height - number.
:param color: Rectangle color, supported formats: #ff0000, rgb(255, 255, 0), argb(255, 0, 0, 0), red.
:param x: Rectangle upper left corner x coordinate on parent.
:param y: Rectangle upper left corner y coordinate on parent.
:param *childs: Primitives rendered inside the rectangle.

Usage:

```
(rectangle {:color "red"
            :width 100
            :height 300
            :x 10
            :y 10}
  (rectangle {:color "#00ff00"
              :x 50
              :y 50
              :x 10
              :y 10}))
```
text
(text {:keys [width height font-size color x y value], :as props} & childs)
Text primitive, can be nested.

:param width: Text holder width - number.
:param height: Text holder height - number.
:param font-size: Font size in px.
:param color: Text color, supported formats: #ff0000, rgb(255, 255, 0), argb(255, 0, 0, 0), red.
:param x: Text holder upper left corner x coordinate on parent.
:param y: Text holder upper left corner y coordinate on parent.
:param value: Text.
:param *childs: Primitives rendered inside the rectangle.

Usage:

```
(text {:width 100
       :height 30
       :font-size 10
       :color "#ff00ff"
       :x 10
       :y 10
       :value "Hi there"}
  (rectangle {:color "#00ff00"
              :x 50
              :y 50
              :x 10
              :y 10}))
```
