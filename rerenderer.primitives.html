rerenderer.primitives

Simple primitives for drawing. Using primitives is more preferd then
creating components by yourself or operating with native-objects.

image

(image {:keys [width height src x y sx sy], :or {width 0, height 0, x 0, y 0, sx 0, sy 0}} & childs)
Image primitive, can be nested:

```
(image {:width 100
        :height 200
        :src "bird" ; `id` of image on bootstraping  html page
        :sx 20 ; x on source image, usable for cutting sprites
        :sy 30 ; y on source image, usable for cutting sprites
        :x 10
        :y 20})
```

rectangle

(rectangle {:keys [width height color x y], :or {width 0, height 0, color [255 0 0 0], x 0, y 0}} & childs)
Rectangle primitive, can be nested:

```
(rectangle {:color [255 0 0 0] ; argb
            :width 100
            :height 300
            :x 10
            :y 10}
  #_ another-rectangle)
```

text

(text {:keys [width height font-size color x y], :or {width 0, height 0, font-size 0, color [255 0 0 0], x 0, y 0}} value & childs)
Text primitive, can be nested:

```
(text {:width 100
       :height 30
       :font-size 10
       :color [255 255 255 0]
       :x 10
       :y 10}
  "Hi there")
```
