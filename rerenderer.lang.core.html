rerenderer.lang.core

Sugar that simplifies writting code that operates with platforms.

This namespace will work only if required like:

(:require [rerenderer.lang.core :as r :include-macros true])

..

macro

(.. x form & more)

Works like clojure .. macro.

Should be used only inside components render methods.

Example:

(r/.. canvas (getContext "2d")) (r/.. image -onLoad (bind (fn [])))

new

macro

(new cls & args)

Works like clojure new macro.

Should be used only inside components render methods.

Example:

(r/new Bitmap)
(r/new Rectangle 100 100 200 200)

set!

macro

(set! path value)

Works like clojure set! macro.

Should be used only inside components render methods.

Example:

(r/set! (r/.. canvas -height) 200)
